# Errors

Stable exception types and error-record mapping helpers used across planning,
execution, extraction, metrics, and storage.

`themis.errors` is the stable namespace for exception imports and
exception-to-record conversion helpers. Import from this root namespace unless
you are
debugging the implementation itself.

::: themis.errors
    options:
      show_root_heading: false

::: themis.errors.exceptions
    options:
      show_root_heading: false

::: themis.errors.mapping
    options:
      show_root_heading: false
