# Guides

Guides are task-oriented. Use them when you know what you need to do and want
the shortest path to it. If you want a lesson with context and checkpoints, go
to [Tutorials](../tutorials/index.md) instead.

- Run a complete, minimal example first: [Example Catalog](examples.md)

- Compare results and export artifacts: [Compare and Export Results](compare-and-export.md)
